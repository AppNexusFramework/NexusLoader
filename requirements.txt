# Installation Instructions

#
# Basic Installation:
#   1. Ensure Python 3.13 is installed
#   2. Create virtual environment: python -m venv .venv
#   3. Activate: 
#      - Windows (PowerShell): .\.venv\Scripts\Activate.ps1
#      - Windows (CMD): .venv\Scripts\activate.bat
#      - Unix/Mac: source .venv/bin/activate
#   4. Upgrade pip: python -m pip install --upgrade pip setuptools wheel
#   5. Install requirements: pip install -r requirements.txt
#
# For automated installation, use the provided scripts:
#   - Windows PowerShell: .\install.ps1
#   - Windows CMD: install.bat
#   - Unix/Linux/macOS: ./install.sh
#
# Cloud Provider Specific (uncomment in file and install):
#   AWS:   boto3, botocore (already included)
#   Azure: azure-storage-blob, azure-servicebus
#   GCP:   google-cloud-storage
#
# Database Specific (uncomment in file as needed):
#   PostgreSQL: psycopg2-binary (already included)
#   MySQL:      pymysql (already included) or mysql-connector-python
#   SQL Server: pyodbc or pymssql
#   Oracle:     oracledb
#   MongoDB:    pymongo
